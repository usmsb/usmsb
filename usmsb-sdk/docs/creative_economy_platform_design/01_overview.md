# 第1章：项目概述与愿景

## 1.1 背景：Z世代创意浪潮与挑战

### Z世代创意人群特征

| 特征 | 描述 | 对平台设计的启示 |
|------|------|------------------|
| 数字原住民 | 将生活融入数字，而非将数字融入生活 | 平台需原生数字化体验 |
| 多元与个性 | 拒绝标签化，追求多元化生活方式和审美 | 需灵活的身份和角色系统 |
| 互动与社群 | 渴望双向沟通，基于共同兴趣形成圈层文化 | 需强大的社交和协作功能 |
| 价值导向 | 重视实际价值和性价比，而非品牌忠诚度 | 需透明的价值评估和分配机制 |

### 创意人群核心痛点

1. **价值冻结**：创意资产难以定价和变现
2. **流动性缺失**：非标资产缺乏交易市场
3. **信任成本高**：陌生人协作缺乏信任基础
4. **议价能力弱**：个体创作者面对B端议价能力不足

### 平台愿景

> 构建新创意经济生态，赋能Z世代创意人群，激活其潜能，实现创意资产的**自由流动**与**价值最大化**。

### USMSB模型

社会行为的通用系统模型，强调核心要素及其动态交互。

| USMSB要素 | 创意经济应用 |
|-----------|-------------|
| **主体** | Z世代创意人群 + AI经济代理 |
| **目标** | 创意实现、价值变现、社群协作 |
| **资源** | 创意作品、技能、时间、数字资产 |
| **客体** | 创意项目、数字版权、服务需求 |
| **规则** | IAP协议、智能合约、社区治理 |
| **信息** | 市场趋势、估值数据、需求匹配 |
| **价值** | 创意变现、效率提升、影响力 |
| **风险** | 版权侵犯、价值低估、协作失败 |
| **环境** | 数字创意市场、Web3生态 |

### IAP协议

IAP是一个**开源的、去中心化的协议层**，旨在成为数字原生经济的基础设施。

**核心原则：**
- **绝对技术驱动**：依靠算法、密码学和物联网实现自动化价值发现
- **服务数字原生代**：为程序员、设计师等群体提供"极客级"工具
- **解决本质问题**：直指价值冻结和流动性缺失的经济学核心矛盾
- **无中介化**：取消传统平台抽佣模式，赋能智能服务体(ISA)

**四大核心功能：**
1. 动态价值发现引擎
2. 流动性碎片化协议
3. zk-信用通行证
4. 联合订单合约

### 雅典娜计划

雅典娜不是中心化的AI平台，而是一个**分布式AI智能体网络**。

**核心特征：**
1. **自主经济代理**：每个创意人拥有AI驱动的"数字分身"
2. **博弈与优化能力**：复杂的多目标优化和博弈决策
3. **网络化协同**：AI代理之间直接通信、协商和达成合作

### 冷启动策略

**问题**：Step 2 和 Step 3 存在循环依赖
- 数据集需要ISA产生交易数据
- ISA需要数据集优化估值模型

**解决方案：**
1. 冷启动：先用模拟数据 + 少量种子用户
2. 激励早期：对早期ISA给予更高激励权重
3. 渐进开放：从特定垂直领域开始（如设计、代码）
