## 1. 整体架构

USMSB SDK 采用分层架构设计，从上到下分为以下层次：

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                              用户层                                          │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐       │
│  │  Web UI     │  │  CLI        │  │  API        │  │  Agent对话   │       │
│  └─────────────┘  └─────────────┘  └─────────────┘  └─────────────┘       │
└─────────────────────────────────────────────────────────────────────────────┘
                                       │
                                       ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                          Meta Agent (超级 Agent)                             │
│  ┌─────────────────────────────────────────────────────────────────────┐    │
│  │                        USMSB Core 集成                                │    │
│  │  ┌───────────────────────────────────────────────────────────────┐  │    │
│  │  │  Agent (九大要素)  │  Goal (目标)  │  Environment (环境)    │  │    │
│  │  │  Resource (资源)   │  Rule (规则)  │  Information (信息)   │  │    │
│  │  │  Value (价值)      │  Risk (风险)  │  Object (对象)        │  │    │
│  │  └───────────────────────────────────────────────────────────────┘  │    │
│  │  ┌───────────────────────────────────────────────────────────────┐  │    │
│  │  │         9 大通用动作 (Universal Actions)                      │  │    │
│  │  │  Perception │ Decision │ Execution │ Interaction │           │  │    │
│  │  │  Transformation │ Evaluation │ Feedback │ Learning │ RiskMgmt │  │    │
│  │  └───────────────────────────────────────────────────────────────┘  │    │
│  └─────────────────────────────────────────────────────────────────────┘    │
│  ┌─────────────────────────────────────────────────────────────────────┐    │
│  │                        Agent Core                                     │    │
│  │  ┌───────────────┐  ┌───────────────┐  ┌───────────────┐          │    │
│  │  │ LLM Manager   │  │ Planner       │  │ Executor      │          │    │
│  │  │ (多LLM支持)   │  │ (任务规划)    │  │ (执行器)      │          │    │

│  │  │ Goal Engine   │  │ Learning      │  │ Wallet        │          │    │
│  │  │ (目标引擎)    │  │ (自主学习)    │  │ (区块链钱包)   │          │    │
│  │  └───────────────┘  └───────────────┘  └───────────────┘          │    │
│  └─────────────────────────────────────────────────────────────────────┘    │
│  ┌──────────────┬──────────────┬──────────────┬────────────────────┐     │
│  │ Tools        │ Skills       │ Memory       │ Knowledge          │     │
│  │ Registry     │ Manager      │ Context      │ Base               │     │
│  └──────────────┴──────────────┴──────────────┴────────────────────┘     │
└─────────────────────────────────────────────────────────────────────────────┘
        │                              │                              │
        ▼                              ▼                              ▼
┌───────────────┐            ┌───────────────┐            ┌───────────────┐
│ System Agents │            │ External      │            │  Storage      │
│ (子Agent蜂群)  │            │ Services      │            │  Layer        │
├───────────────┤            ├───────────────┤            ├───────────────┤
```

## 2. 模块划分

### 2.1 核心模块

| 模块 | 路径 | 职责 |
|------|------|------|
| **agent_sdk** | `src/usmsb_sdk/agent_sdk/` | Agent 开发工具包 |
| **core** | `src/usmsb_sdk/core/` | 核心逻辑（技能、通信、通用动作） |
| **intelligence_adapters** | `src/usmsb_sdk/intelligence_adapters/` | 智能适配器（LLM、知识库） |
| **platform** | `src/usmsb_sdk/platform/` | 平台功能 |

### 2.2 协议层

| 模块 | 路径 | 职责 |
|------|------|------|
| **protocol/a2a** | `src/usmsb_sdk/protocol/a2a/` | Agent-to-Agent 协议 |
| **protocol/mcp** | `src/usmsb_sdk/protocol/mcp/` | Model Context Protocol |
| **protocol/websocket** | `src/usmsb_sdk/protocol/websocket/` | WebSocket 通信 |
| **protocol/p2p** | `src/usmsb_sdk/protocol/p2p/` | P2P 通信 |
| **protocol/grpc** | `src/usmsb_sdk/protocol/grpc/` | gRPC 通信 |
| **protocol/http** | `src/usmsb_sdk/protocol/http/` | HTTP 通信 |

### 2.3 服务层

| 模块 | 路径 | 职责 |
|------|------|------|
| **services** | `src/usmsb_sdk/services/` | 核心服务 |
| **node** | `src/usmsb_sdk/node/` | 节点功能 |
| **data_management** | `src/usmsb_sdk/data_management/` | 数据管理 |
| **logging_monitoring** | `src/usmsb_sdk/logging_monitoring/` | 日志监控 |
| **reasoning** | `src/usmsb_sdk/reasoning/` | 推理引擎 |

## 3. 核心组件

### 3.1 Meta Agent

Meta Agent 是整个系统的核心，具备：

- **九大通用动作实现**
- **多LLM支持**
- **动态技能系统**
- **智能记忆系统**
- **区块链钱包**
- **自主学习进化**

### 3.2 LLM 适配器

支持多种 LLM 提供商：

- OpenAI
- Anthropic (Claude)
- Google Gemini
- 智谱 GLM
- MiniMax
- 本地模型

### 3.3 工具注册表

动态注册和管理工具：

```python
class ToolRegistry:
    def register(self, tool: Tool): ...
    def get_tool(self, name: str): ...
    def list_tools(self): ...
    def execute(self, name: str, **kwargs): ...
```

## 4. 数据流

```
用户输入
    │
    ▼
┌─────────────┐
│ 感知层      │ ← 解析输入、提取意图
└─────────────┘
    │
    ▼
┌─────────────┐
│ 推理层      │ ← 逻辑推理、因果分析
└─────────────┘
    │
    ▼
┌─────────────┐
│ 规划层      │ ← 任务分解、路径规划
└─────────────┘
    │
    ▼
┌─────────────┐
│ 执行层      │ ← 调用工具、执行动作
└─────────────┘
    │
    ▼
┌─────────────┐
│ 评估层      │ ← 评估结果、质量检查
└─────────────┘
    │
    ▼
┌─────────────┐
│ 学习层      │ ← 知识积累、能力提升
└─────────────┘
    │
    ▼
  用户输出
```
